Web Shell 脚本检测

本项目提供 PHP Web Shell 的危害分析，基于朴素贝叶斯、深度神经网络与卷积神经网络三种方法。

项目说明

所用的有害 PHP 脚本列表，均来自于 Github：

  • bartblaze：PHP-backdoors
  • BlackArch：webshells
  • dotcppfile：DAws
  • JohnTroony：php-webshells
  • mattiasgeniar：php-exploit-scripts
  • tanjiti：webshellSample
  • tdifg：Webshell
  • tennc：webshell
  • testsecer：WebShell
  • xl7dev：WebShell
  • ysrc：webshell-sample

仅供学习和模型测试用，切勿用于其他用途！

所用的正常 PHP 脚本列表，均来自于 Github 高星开源项目：

  • CodeIgniter
  • laravel
  • phpmyadmin
  • symfony
  • typecho
  • WordPress
  • yii2
